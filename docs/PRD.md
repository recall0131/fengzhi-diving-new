# 奉旨潜水全球双语官网 PRD
## 项目定位
全球专业潜水服务平台，主营潜水装备代理、定制行程、认证课程，以线索转化为核心业务，支持中英文双语访问，为潜水爱好者、旅行社、企业客户提供一站式服务。

## 目标用户
- 中文用户：国内潜水爱好者、线下潜水俱乐部、企业团建/旅行采购
- 英文用户：海外华人潜水群体、外籍潜水员、国际潜水旅行社

## 成交路径
首页 → 产品/行程/课程栏目 → 详情页 → 线索表单/在线咨询 → 销售跟进转化

| 路径（zh/en） | 页面功能 |

## 后台需求与权限
- 内容管理：增删改查产品/行程/课程/博客文章
- 线索管理：查看所有提交的表单数据，导出Excel
- 用户权限：管理员（全权限）、编辑（仅内容管理）
- 系统配置：SEO元信息、站点设置

## SEO策略
1. 双语URL本地化：`/zh/xxx` / `/en/xxx`
2. 页面级meta标签：标题、描述、关键词（多语言适配）
3. 结构化数据：Schema.org 产品/行程/课程标记
4. 图片alt文本：多语言适配
5. 自动生成sitemap.xml / robots.txt
6. 静态资源压缩与懒加载，优化全球访问速度
