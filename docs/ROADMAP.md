# 奉旨潜水官网 开发路线图
## MVP阶段（2-4周）
- 基础框架搭建：Next.js + NestJS + PostgreSQL
- 双语路由与i18n国际化
- 核心页面开发：首页/产品/行程/课程/联系页
- 线索表单提交功能
- Directus CMS 内容管理系统部署
- 基础SEO配置与静态资源优化
- Docker Compose 一键部署脚本

## 增强阶段（4-8周）
- 博客文章系统开发
- 站内搜索功能
- 邮件通知：表单提交成功通知、管理员提醒
- 监控与日志系统集成
- 全球CDN配置优化访问速度
- 多角色权限管理完善

## 商业化阶段（8-12周）
- 支付集成（可选：微信/支付宝/Stripe）
- 会员系统与积分功能
- API开放平台：对接第三方旅行社/装备供应商
- 数据分析面板：用户行为、转化漏斗
- 自动化营销工具：邮件营销、短信提醒
