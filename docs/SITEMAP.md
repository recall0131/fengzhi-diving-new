## 关键页面线框（文字版）
1. **首页**：
   - 导航栏：品牌LOGO + 产品/行程/课程/关于/联系 + 双语切换按钮
   - Hero区：品牌 slogan + 主CTA按钮（立即咨询）
   - 核心业务板块：装备/行程/课程 卡片式展示
   - 线索表单：姓名/电话/需求类型 快速提交
   - Footer：联系方式、版权、社交媒体链接

2. **列表页**：
   - 筛选栏：分类/价格/地区 筛选条件
   - 卡片列表：缩略图、标题、价格、简短描述
   - 分页/加载更多组件

3. **详情页**：
   - 产品图片轮播
   - 详细参数/行程/课程内容
   - 咨询表单：姓名/电话/留言
   - 相关推荐板块

4. **后台仪表盘**：
   - 线索统计图表
   - 内容管理快捷入口
   - 系统状态监控
